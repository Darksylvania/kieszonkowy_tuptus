# Kieszonkowy Tuptuś

Kiedy Tuptuś hopsa za daleko, żeby usłyszał

A fun and user-friendly Progressive Web App (PWA) for converting between various units of measurement. Convert length, weight, volume, area, and more with ease!

## Features

- **Unit Conversion**: Convert between multiple categories including length, weight, volume, and area
- **Area Calculator**: Calculate rectangle area with different unit combinations
- **Proportion Calculator**: Solve ratios in the form A:B = C:D
- **Collapsible Sections**: Clean, organized interface with expandable sections
- **PWA Support**: Install as a standalone app on mobile devices
- **Offline Capable**: Works offline thanks to service worker caching
- **Responsive Design**: Optimized for desktop and mobile

## Technologies Used

- HTML5
- CSS3 (with custom properties and animations)
- Vanilla JavaScript
- Progressive Web App (PWA) features

## Installation

### Web Version
Simply open `index.html` in a modern web browser.

### PWA Installation
1. Open the app in a supported browser (Chrome, Edge, Safari)
2. Click the install button in the address bar or use the browser menu
3. The app will be installed as a standalone application

### Android APK
Use tools like PWABuilder or Capacitor to generate an APK from the PWA.

## Usage

1. Select a category (Length, Weight, Volume, Area)
2. Choose source and target units
3. Enter a value to convert
4. Use the calculators for area or proportions as needed

## Development

To run locally with a development server:

```bash
npm start
```

This starts a simple HTTP server on port 8080.
